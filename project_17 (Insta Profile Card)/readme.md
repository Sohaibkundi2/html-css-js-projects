# Instagram Profile Card

This project is a simple, interactive profile card inspired by Instagram's design. It features a user's profile picture, name, and a status that changes dynamically. The main interactive element is an "Add Friend" button that toggles to "Friend" when clicked.

## Features

- **Custom Cursor:** A circular cursor that follows the mouse for a unique user experience.
- **Interactive Button:** An "Add Friend" button that changes to "Friend" on click, with corresponding color and text changes.
- **Dynamic Status:** The user's status updates in response to the button click.
- **GSAP Animations:** Utilizes GSAP for smooth animations and visual effects.
