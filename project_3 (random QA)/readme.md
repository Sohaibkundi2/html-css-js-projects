# Random Question Answer Generator

This is a simple and interactive math quiz web application built using HTML, CSS, and JavaScript. The app dynamically generates random math questions and checks the user's answers in real-time, updating the score accordingly.

## Features

- Generates random math questions on:
  - Multiplication
  - Addition
  - Subtraction
  - Division
- Real-time score tracking.
- Immediate feedback with styled messages:
  - Green banner for correct answers
  - Red banner for incorrect answers
- Smooth animations and glassmorphism-based UI design.
- Responsive layout with clean interface.

## Technologies Used

- HTML5
- CSS3 (Animations, Glassmorphism UI)
- JavaScript (DOM Manipulation, Math Logic)


## How It Works

1. A random math question is generated on page load or after each answer.
2. The user inputs their answer and submits it.
3. If the answer is correct:
   - The score increases by 1.
   - A success banner appears.
4. If the answer is incorrect:
   - The score decreases by 1.
   - An error banner appears.
5. A new question is generated randomly each time.
